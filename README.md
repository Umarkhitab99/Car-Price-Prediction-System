# Car-Price-Prediction-System :
A Machine Learning project built with Python that predicts car prices based on manufacturing year and kilometers driven using the Linear Regression algorithm. The project demonstrates data preprocessing, model training, prediction, and data visualization using Pandas, NumPy, Matplotlib, and scikit-learn.
Features:
 Data handling using Pandas
 Numerical operations with NumPy
 Data visualization using Matplotlib
 Machine Learning model using scikit-learn
 Train/Test data splitting for model evaluation
 Predicts car prices from user-defined input values
Technologies Used :
 Python
 NumPy
 Pandas
 Matplotlib
 Scikit-learn
Machine Learning Algorithm :
 The project uses:
 Linear Regression
 Linear Regression is a supervised machine learning algorithm used to predict continuous numerical values.
 In this project, it predicts the car price using:
 Car Manufacturing Year
 Kilometers Driven
Project Workflow :
 Import required libraries
 Create and load dataset
 Prepare input and output variables
 Split dataset into training and testing data
 Train the Linear Regression model
 Predict car prices
 Visualize the dataset using graphs
Example Prediction :
 Input:
 Year = 2022
 KM Driven = 5000
 Output:
 Predicted Car Price
Learning Outcomes :
 This project helps beginners understand:
 Data preprocessing
 Feature selection
 Model training
 Prediction using Machine Learning
 Data visualization
 Supervised Learning concepts
